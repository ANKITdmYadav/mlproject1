## Student Performance Dataset
This dataset contains information about students’ demographic background, parental education levels, test preparation status, and their scores in math, reading, and writing.
It can be used for data analysis, visualization, machine learning, and educational performance research.

#Dataset Description
The dataset includes 8 features describing each student and 3 numerical scores representing their academic performance.

| Column Name                     | Description                                       |
| ------------------------------- | ------------------------------------------------- |
| **gender**                      | Gender of the student (e.g., male, female)        |
| **race_ethnicity**              | Student’s ethnic group (A, B, C, etc.)            |
| **parental_level_of_education** | Highest education level of the parent(s)          |
| **lunch**                       | Type of lunch received (standard or free/reduced) |
| **test_preparation_course**     | Whether the student completed a test prep course  |
| **math_score**                  | Score obtained in mathematics                     |
| **reading_score**               | Score obtained in reading                         |
| **writing_score**               | Score obtained in writing                         |
